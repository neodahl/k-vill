UI Guide

UI 스타일은 기본적으로 "kebab-case" 로 클래스를 작성되었습니다. 간혹, "camelCase" 를 사용한 스타일이 있다면 이는 퍼블리싱 스크립트와 연동되는 클래스임을 암시합니다.
개발용 클래스가 필요하다면, 명확한 구분을 위해 반드시 "PascalCase" 로 사용해주시길 바랍니다.

Buttons

필요에 따라 "button" 또는 "a" 태그에 CSS class를 조합하여 사용합니다.
클래스의 시작은 반드시 "btn-" 로 시작해야하며,
사이즈 에 따라. "-sm, -md, -lg" 추가 (전체 너비는 "-full" )
타입 에 따라 "-bg, -line" 을 추가
색상 에 따라 "-primary, -secondary" 추가
윤곽 에 따라 "-round" 추가
접두사("btn-")이외에 나머지 접미사는 순서에 상관 없이 사용할 수 있습니다.
ex) "btn-sm-bg-primary" == "btn-primary-sm-bg"
권장 사용 방식은 "btn-사이즈-타입-색상" 입니다.
경우에 따라 "-사이즈", "-색상" 등은 생략될 수 있습니다.
